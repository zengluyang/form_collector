# form_collector
统计表单利器

A simple form collector.

Yii,Bootstrap and jQuery are used

Database setup in `protected/config/main.php`

```
		'db'=>array(
			'connectionString' => 'mysql:host=localhost;dbname=form',
			//'emulatePrepare' => true,
			'username' => 'root',
			'password' => 'password',
			'charset' => 'utf8',
			//'emulatePrepare' => false,
		),
```
